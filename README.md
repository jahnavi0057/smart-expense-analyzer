Smart Expense Analyzer
Overview

An AI-powered application that analyzes financial transactions, categorizes expenses, and provides insights and predictions to help users manage spending effectively.

Features
Upload CSV expense data
Automatic categorization (Food, Transport, Bills, Shopping, Others)
Spending analysis by category
Data visualization (bar charts)
Expense prediction using Linear Regression
Smart insights (overspending alerts)
Tech Stack
Python
Streamlit
Pandas
Scikit-learn
Matplotlib
How to Run

pip install -r requirements.txt
streamlit run app.py

Use Case

Helps users understand spending behavior and make better financial decisions using AI-driven insights.
